## 🔗 [View Live Site]tps://adithravi.github.io/job-portal-

### The Goal
Most engineering job boards are cluttered. I built this to practice creating a **frictionless user experience** where students can find roles in AI, Design, and Mechanical Engineering in seconds.

### ⚡ Key Features
* **Live Search:** Find roles instantly—no page reloads.
* **Clean Design:** A card-based layout optimized for mobile and desktop.
* **Smart Filtering:** Jobs are categorized by department for easy scanning.

### 🛠️ Built With
* **Languages:** HTML5, CSS3, JavaScript (ES6)
* **Design Principles:** Focused on visual hierarchy and scannability.
